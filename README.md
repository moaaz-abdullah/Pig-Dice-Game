# 🎲 Pig Dice Game

A fun and interactive **two-player dice game** built with **JavaScript, HTML, and CSS** as part of Jonas Schmedtmann's JavaScript course.

---

## 🚀 Overview

This is a classic **Pig Game** where two players compete to reach **100 points first**.

* Players take turns rolling a dice 🎲
* Each roll adds to their **current score**
* Rolling a **1 resets the current score** and switches the turn
* Players can choose to **Hold** to save their score
* First player to reach **100 points wins** 🏆

---

## 🧠 Game Rules

1. Roll the dice to accumulate points
2. If you roll:

   * **2–6** → added to current score
   * **1** → lose current score & switch turn
3. Click **Hold** to save your score
4. First player to reach **100 points wins**

---

## 🛠️ Features

* Two-player gameplay
* Random dice generation 🎲
* Dynamic UI updates using DOM manipulation
* Game state management (scores, turns, winner)
* Reset game functionality
* Visual feedback for active player and winner
